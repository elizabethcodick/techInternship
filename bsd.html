What is BSD?

Basel Scorecard Database (BSD) is a system that is used at M&T to create, manage, view and store scorecards that grade financial risk for the bank. The scorecards calculate Probability of Default (PD) and Loss Given Default (LGD).

Current User Interface (UI)

The main focus was to evaluate the current BSD software and redesigning the screens to have a more modern look and feel. The evaluation including analyzing the software for pain points and figuring out the best way to get rid of them.

Redesign of the UI

After the evaluation of the old UI several paper and pencil prototypes were made and then converting them into digital wireframes using Ninjamock (online program). Throughout the process there were several meetings with the business partners, the people who use the software, and the development team.
